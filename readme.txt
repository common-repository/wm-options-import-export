Wordpress Options Import/Export. Backup Settings.

== Description ==

"Options Import Export" is a plugin which allows Export Wordpress Options(Settings) and Import Wordpress Options(Settings)

If you have an interest in customization this plugin please let me know.

== Installation ==
1. Install the WM Options Import Expor
2. Activate the plugin
3. Wordpress Admin > Tools > Export Options
4. Wordpress Admin > Tools > Import >  Wordpress Options
